## Technical Specifications

### Output Format
- **Resolution**: 1024x1024 or 512x512 (for web optimization)
- **Duration**: 3-5 seconds
- **Frame Rate**: 24-30 fps
- **File Size**: Under 5MB for web, under 15MB for high quality
- **Loop**: Seamless (first and last frame match)

### Recommended Tools
- **Runway Gen-2/Gen-3** — Best for cinematic quality
- **Pika Labs** — Good for stylized animations
- **Midjourney + img2vid** — For static-to-animated
- **Stable Video Diffusion** — Open source option
- **After Effects + Lottie** — For vector-based reveals

### Brand Alignment Checklist
- [ ] Uses brand color palette (gold, burgundy, warm black)
- [ ] Victorian/Gilded Age aesthetic
- [ ] Warm, candlelit lighting
- [ ] Elegant, unhurried motion
- [ ] Subtle particle effects (dust, light)
- [ ] High production value feel
- [ ] Seamless loop capability
